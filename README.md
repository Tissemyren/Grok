# Grok
This is a small Discord user installable app that allows you to click "grok is this true" on any message and get a response from Grok regarding if said message is factual.
The bot actually prompts the Gemini API, as I do not intend to put money into an xAPI subscription..

Besides a basic fact checking feature, this bot also has a button to add context to prompts and a button to extract and translate text from images.

# Grok
This is a small Discord user installable app that allows you to click "grok is this true" on any message and get a response from Grok regarding if said message is factual.
(It actually uses Gemini because I didn't wanna pay for Grok...)

I added features to add context to prompts via a Discord Modal before sending it, and a context menu to translate images.
As this bot is not exactly intended for professional use, the image translation is not sufficient if you want accurate translations, as the implementation of the text extraction is pretty basic.

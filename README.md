# Grok
This is a small Discord user installable app that allows you to click "grok is this true" on any message and get a response from Grok regarding if said message is factual.
(It actually uses Gemini because I didn't wanna pay for Grok...)

Went all in and added more stuff tho - like fact checking with extra, user-defined, context, and even image translation.
The library for extracting text from images is not exactly the best, so don't expect great results, alright?
